# Ohm-s-law-calculator

### Overview

This project provides you with a calculator built in C that solves ohm's law ie

**V = I × R**

### How it is useful

This is the most basic yet most important equation in circuit analysis .Instead of manual calculation, this tool does it instantly — useful during lab sessions, assignments, or quick checks.


### What can it calculate

* Voltage (V)
* Current (I)
* Resistance (R)
* Power(P)

### Key features:

* Menu-driven interface
* Handles errors
* Loop
